inSight

A Deep Neurofeedback Model

Last summer (2019), I had the opportunity to join Professor Pattie Maes and PhD candidate Neo Mohsenvand at MIT's Media Lab (Fluid Interfaces Group). I worked on inSight, a brain decoding system that uses EEG data to train deep neural networks and recreate what images a user was seeing.

My role in this project was to conduct research on the cognitive aspects of the project and program the stimuli. I used Jupyter Notebook (language: Python) to generate various visual stimuli (videos). I also assisted in creating a deep neural network and worked with two different generative adversarial networks (StyleGan and BigGan).

For more information, please visit inSight Project on the Media Lab's page. To view the code for the stimuli I worked on, visit this Github page.
