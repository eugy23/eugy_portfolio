NYC Taxis

Data Visualization Using R

For my final project in CS100: Data Fluency, I, along with two other Brown University students, looked at a public data set provided by the NYC Taxi Limousine Commission. We took 1% of the taxi rides in New York City in 2014 (1.6 observations and over 20 variables) to create some interesting visualizations and observations.

This project was done using R. For the full code and visualizations, visit our project page.
